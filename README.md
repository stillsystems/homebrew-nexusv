## Overview

This repository contains Homebrew formulae for Still Systems tools, specifically **Nexus-V**. It ensures that macOS and Linux users can easily install and update our tools using the `brew` package manager.

## Installation / Quickstart

To add this tap to your Homebrew installation:

```bash
brew tap stillsystems/nexusv
```

Then you can install tools as usual:

```bash
brew install nexus-v
```

## Usage

Once installed, you can keep your tools up to date using:

```bash
brew update
brew upgrade nexus-v
```
